# 🧪 Institute Experiments (frontend)

Frontend-часть приложения по работе с данными по различным экспериментам

🔗 **[Backend-часть находится тут](https://github.com/coolrinet/institute-experiments-backend)**

Проект был создан в рамках написания выпускной квалификационной работы для кафедры машин и аппаратов промышленных технологий Сибирского государственного университета науки и технологий имени академика М. Ф. Решетнёва.

---

## 🚀 Возможности приложения

### 👤 Для пользователей

- Добавление/редактирование/удаление установок, на которых проводятся эксперименты
- Добавление/редактирование/удаление параметров, которые могут быть общими, так и принадлежать какой-то конкретной установке
- Создание исследований, где будут проводится все эксперименты на определенной установке
- Изменение видимости исследования (публичное или доступное только определенным людям)
- Добавление/изменение/удаление экспериментов, проведенных на установке исследования
- Экспорт данных исследования в виде CSV

### 🛡️ Для администраторов

Такие же, как и у обычного пользователя, плюс:

- Добавление/удаление нового пользователя (добавленным пользователям приходит письмо на электронную почту)
- Изменение прав пользователя (возможность сделать его администратором)

---

## ⚙️ Стек технологий

**Frontend**
- React 18
- Typescript - типизация
- Tanstack Query - работа с асинхронным состоянием
- Tanstack Router - маршрутизация
- React Form Hook - работа с формами
- Zod - валидация данных
- Mantine v7 - UI-библиотека

**Backend**
- PHP 8.3
- Laravel 11
- Laravel Sanctun - аутентификация пользователей

## Локальный запуск (frontend)

```bash
pnpm install
```

Задай переменную окружения `VITE_APP_URL` (адрес API) в .env, после чего

```bash
pnpm dev
```
